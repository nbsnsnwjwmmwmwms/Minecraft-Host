# Minecraft-Host
MadHost - its local server host

How to use:
1. Dowandload MineHost in realeses
2. Unpack to your computer
3. open Service.bat in edit mode
4. change Max and Min selection operating memory
5. Win + R
6. Write cmd
7. in cmd write ipconfig
8. copy Ipv4 IP
9. open Server.propetines
10. in line "Server-adress" write IP
11. Start Server.cmd

AFTER UNPACKING MINEHOST:
Open server.bat to download Spigot 1.12.2
